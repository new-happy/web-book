注意:

  1. react 组件内部有一个默认参数叫 props 作用是接收父组件传过来的参数 如果是函数组件 函数第一个参数默认就是props 如果是es6 class组件的话组件内部用 this.props获取props
  2. props对象是只读的 不能进行修改 当父组件传给子组件一个react节点的时候,子组件使用this.props.children接收
